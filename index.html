                                                 YOCTO Project

I choose YOCTO project for this assignment. YOCTO project is an open source project that offers templates, tools and method to help user to create custom Linux-based systems for implant products. It was founded in 2010 as a teamwork among many hardware manufactures, open source operating systems vendors and other electronics companies some order to the chaos of embedded Linux development. The purpose of this project that it’s a complete implanted Linux development environment with tools, metadata, and documentation that people need. The YOCTO project allow its users to focus on their particular product features and development. This project produce resources and information delivers to new and experience users. It also provides pointers to example code built indicating its capabilities.

The YOCTO project sponsor by Linux Foundation. Everyone can participate in this project. There are the participants such as NXP, Wind River, OpenEmbedded eV, Texas Instruments, Intel, Long Term Support Initiative-LTSI, Juniper Networks, AMD and Mentor Graphics for Gold member. The Silver Member sponsors are Linaro, Timesys, Broadcom, LG, Dell, Reness, OS Sysytems, MontaVista Software LLC, Huawei and Enea. The governance structure is based on the best practices and values of Open Source culture, and pursues the goals set out in the Yocto Project Charter. The YOCTO Project is open to all contributors because it doesn’t requires no admission processes, contracts, or membership fees, any individual or organization can use and contribute to the project. The YOCTO projects welcome all the participants that lead to the success of the project including software development, documentation, testing and advocay.

In order to contribute on the project users need to download the ADT and Toolchains. One of the processes of YOCTO is to download the Tarball, build the Tarball and then run it. These are the codes that user need to run the data: 

 $ cd ~

 $ git clone git://git.yoctoproject.org/poky

 $ cd poky

 $ git checkout -b jethro origin/jethro

 $ source oe-init-build-env

 $ bitbake adt-installer
